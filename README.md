### 概览

欢迎来到《鄞年・思叙：鄞州中学校园期刊》（*Yinzhou High School Campus Journal*）第二期的官方 GitHub 仓库，这里将是我们未来工作流的中心。

### 为什么要建立此仓库？

我们原先用 [Craft](https://www.craft.do/) 跑通了全流程，但为了本期更大规模的编辑部的顺利运转，也为了将我们的幕后工作流程同步、展现给各位，特此建立该仓库。

### 我们的工作流程

- [x] 立项（确定档期、《征稿说明》）
- [ ] 征稿
- [ ] 改稿
- 封面照片拍摄
- 校方稿件审核
- 统稿
- 封面制作
- 排版
- 校方成品审核
- 电子版发布
- 印刷
- 校内纸质版发放

注：已勾选的复选框代表已完成，待勾选的复选框代表进行中，普通的项目符号代表未开始。
